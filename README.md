# WJPhotoBrowser
* An easy way to browse image
* 用法简单的图片浏览器，可以浏览网络图片和本地图片

## Contents
* Getting Started
    * [System 【iOS7+】]

## <a id="如何使用WJPhotoBrowser"></a>如何使用WJPhotoBrowser
    * cocoapods导入：`pod 'WJPhotoBrowser'` (努力实现中)
    * 手动导入：
    * 将`WJPhotoBrowser`文件夹中的所有文件拽入项目中
    * 导入主头文件：`#import "WJPhotoBrowser.h"`
